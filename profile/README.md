<h2>What is WalletPlug</h2>
<p>
WalletPlug is a fully registered payment gateway and modern fintech infrastructure platform that enables developers and SaaS founders to onboard easily and accept payments locally and internationally. We handle the complexity of payments so you can focus on building features instead of wiring financial infrastructure.
</p>

<h2>Products and Integrations</h2>
<ul>
  <li><strong>WHMCS Plugin</strong> for billing hosting and subscriptions</li>
  <li><strong>WordPress Plugin</strong> for website payments and checkout</li>
  <li><strong>Shopify Plugin</strong> for ecommerce payments and order processing</li>
  <li><strong>Mobile SDKs</strong> for Android and iOS payment experiences</li>
  <li><strong>APIs and Webhooks</strong> for payments wallets cards payouts and automation</li>
</ul>
